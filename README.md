# TravelPlanner

A simple website project for planning group travels.

## Files
- `V4.html`: Main HTML file for the website
- `V4.css`: Stylesheet for the website
- `V4.js`: JavaScript for interactive features

## How to use
1. Open `V4.html` in your browser to view the website.
2. Edit `V4.css` to change the look and feel.
3. Edit `V4.js` to add or modify interactive features.
